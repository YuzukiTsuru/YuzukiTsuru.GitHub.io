使用`binascii`进行转换。


```python
import binascii
import serial

def order_list(hex='00000000'):
    hex_list = []
    for i in hex.split():
        hex_list.append(binascii.a2b_hex(hex))
    return hex_list

ser = serial.Serial('/dev/ttyAMA0', 115200)
ser.writelines(order_list('770000000000ee'))
```
