1.首先更新一下密钥，看看能不能生成自动修复。如果不能直接下一步。（如果没有安装archlinux-keyring，请及时安装）

```bash
sudo pacman-key --refresh-keys
```

2.重新加载相应的签名密钥

```bash
sudo pacman-key --init
sudo pacman-key --populate
```

3.清除`pacman`的缓冲文件

```bash
sudo pacman -Scc
```

4.更新或者安装即可

```bash
sudo pacman -Syu
```
