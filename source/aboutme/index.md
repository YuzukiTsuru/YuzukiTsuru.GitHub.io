## 主要技术方向

### 嵌入式开发

- Tina Linux SDK 开发与适配
- SyterKit Bootloader 开发
- 主线 Linux 内核适配
- HIFI5 DSP 开发
- NPU/OpenCV 应用

### Linux / 操作系统

- Linux 内核开发与调试
- U-Boot Bootloader
- BuildRoot 构建系统
- GCC 工具链
- 各类 Linux 发行版 (Manjaro、Debian)

### RISC-V 架构

- RISC-V ISA 基础架构
- D1/D1s 等国产 RISC-V 芯片开发
- RISC-V Debian 系统构建

### 硬件设计

自行设计开发板项目：

| 项目名称 | 平台 | 说明 |
|----------|------|------|
| Yuzuki ESP32 SOLO | ESP32 | 单板设计 |
| Yuzuki2040 | RP2040 | Raspberry Pi RP2040 开发板 |
| YuzukiXR806 | XR806 | 全志 XR806 开发板 |
| TinyVision 系列 | V85x | 带摄像头的小型视觉开发板 |

涉及 PCB 设计、电路原理图、EE 硬件调试。

### 歌声合成

- DSP 音频处理
- UTAU 相关开发
- **lessampler** — C++ 歌声合成项目

### C++ 开发

- JUCE 音频应用框架
- Libtorch 机器学习库
- QT/QML 界面开发
- CMake 构建系统
- Emscripten WebAssembly
